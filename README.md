# My Personal Portfolio

Welcome to the source code for my personal portfolio website. This is where I introduce myself, showcase my technical skills, and feature the projects I've passionately built.

**Live Demo:** [Your-Vercel-Deployment-Link.com]


## Key Features

The website is structured into several key sections:

* **Home:** A brief welcome and introduction to make a strong first impression.
* **About:** More detailed information about my background, education, work experience, and career goals.
* **Skills:** A showcase of the technologies, programming languages, and tools I'm proficient in.
* **Projects:** A gallery of my notable projects and links (to live demos and repositories).

## Tech Stack

This project was built using a modern set of technologies focused on performance and an excellent developer experience:

* **Language:** TypeScript
* **Framework:** React.js
* **Styling:** Tailwind CSS
* **UI Library:** shadcnui
* **Icons:** Lucide React
* **Font:** Geist Font
* **Deployment:** Vercel

## Deployment

This project is configured for continuous integration and deployment (CI/CD) using **Vercel**. Any push or merge to the `main` branch will automatically trigger a new build and deploy the latest version of the site.
